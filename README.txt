Эта программа создана для "скрабинга" веб-сайтов.

Для работы Вам необходимо написать в терминале следующее: 
"python app.py <link> <link2> ... <link..> ",
где <link> ссылка/ссылки

Программа может обработать несколько сайтов за раз, если Вы передадите ей в качестве аргументов соответсвующие значения.

Пример запуска программы:
"python app.py https://lenta.ru/news/2017/07/16/federercilic/"

В следуюшем примере происходит работа с двумя сайтами:
"python app.py https://lenta.ru/news/2017/07/16/federercilic/ http://www.bbc.com/russian/vert-fut-40647338"
