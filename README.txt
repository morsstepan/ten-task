Эта программа создана для "скрабинга" веб-сайтов.
Для работы Вам необходимо написать в терминале следующее: "python app.py <link> <link2> ... <link..> <lang>",
где <link> ссылка, а <lang> язык в виде "ru", "en" ...
Например, "python app.py https://lenta.ru/news/2017/07/16/federercilic/"
В данном примере параметр <lang> опускается. Если его не указать, то программа автоматически устанавливает русский язык для веб-сайта, с которого происходит "скабинг".
В следуюшем примере происходит работа с англоязычным сайтом:
"python app.py http://www.bbc.com/news/business-40658774 en"
